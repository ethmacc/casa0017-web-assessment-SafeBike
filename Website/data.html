Data Sources

The project primarily utilized data from the MPS Recorded Crime, detailed by Lower Layer Super Output Areas (LSOAs) in London, obtained from the Metropolitan Police via the London Datastore. LSOAs, containing 400-1200 households and 1000-3000 residents, offer detailed geographic granularity compared to larger areas like boroughs. This dataset included crime incident counts across different types for each LSOA over two years, necessitating preprocessing before website integration.

Data Processing

Initial steps involved using a Jupyter notebook and Python's Pandas library to filter out irrelevant crime data. Additionally, LSOA boundary coordinates were absent and had to be sourced from the London Datastore, integrated using Geopandas, and merged with the MPS data. Precomputed totals for each LSOA were added to a new column to minimize dynamic processing. The final processed data was exported as a .geojson file.
